## 二、标题 (headings)

标题界面分为两个功能区：**标题编号规则**（左侧/上部）和 **标题格式设置**（右侧/下部/分Tab页）。

### 2.1 标题编号规则（Heading Numbering Rules）

> 对应 JSON 路径：`patterns.rules[0..3]`，4条规则

界面形态：表格/列表，每行一条规则，列字段如下：

| 前端控件 | JSON字段 | 类型 | 示例值 | 说明 |
|---------|----------|------|--------|------|
| 启用复选框 | `rules[n].enabled` | bool | `true` | 是否启用该编号规则 |
| 编号方案下拉框 | `rules[n].scheme` | string | `"ZH_NUM"` / `"ARABIC"` | 编号数字方案（中文数字/阿拉伯数字） |
| 包装器下拉框 | `rules[n].wrapper` | string | `"DUNHAO"` / `"DOUBLE_PAREN"` / `"DOT"` / `"SINGLE_PAREN"` | 编号后缀/括号样式 |
| 多层深度输入框 | `rules[n].multi_depth` | string | `"1.1"` / `"1.1.1"` / `"1.1.1.1"` | 多级编号深度模式 |
| 自定义示例文本框 | `rules[n].custom_example` | string | `""` | 自定义编号示例预览 |

4条默认规则：

| 规则 | 编号方案 | 包装器 | 多层深度 | 效果示例 |
|------|---------|--------|---------|---------|
| 规则1 | 中文数字(`ZH_NUM`) | 顿号(`DUNHAO`) | `1.1` | 一、→ (一)→ 1. → (1) |
| 规则2 | 中文数字(`ZH_NUM`) | 双括号(`DOUBLE_PAREN`) | `1.1` | (一)→ (二)→ (三) |
| 规则3 | 阿拉伯数字(`ARABIC`) | 点(`DOT`) | `1.1.1` | 1.1 → 1.1.1 |
| 规则4 | 阿拉伯数字(`ARABIC`) | 单括号(`SINGLE_PAREN`) | `1.1.1.1` | 1) → (1) → ① |

| 字号下拉框 | `headings[n].size_cn` | string | `"三号"` | `"小三"` | `"四号"` |
| 粗体复选框 | `headings[n].bold` | bool | `true` | `true` | `true` |
| 斜体复选框 | `headings[n].italic` | bool | `false` | `false` | `false` |
| 下划线复选框 | `headings[n].underline` | bool | `false` | `false` | `false` |
| **行距区** | | | | | |
| 行距模式下拉框 | `headings[n].line_spacing_mode` | string | `"EXACT"` | `"EXACT"` | `"EXACT"` |
| 行距值输入框 | `headings[n].line_spacing_value` | float | `26.0` | `26.0` | `26.0` |
| **段落间距区** | | | | | |
| 段前间距输入框 | `headings[n].space_before_value` | float | `0.0` | `0.0` | `0.0` |
| 段前单位下拉框 | `headings[n].space_before_unit` | string | `"行"` | `"行"` | `"行"` |
| 段后间距输入框 | `headings[n].space_after_value` | float | `0.0` | `0.0` | `0.0` |
| 段后单位下拉框 | `headings[n].space_after_unit` | string | `"行"` | `"行"` | `"行"` |
| **缩进区** | | | | | |
| 左缩进输入框 | `headings[n].left_indent_value` | float | `0.0` | `0.0` | `0.0` |
| 左缩进单位下拉框 | `headings[n].left_indent_unit` | string | `"字符"` | `"字符"` | `"字符"` |
| 右缩进输入框 | `headings[n].right_indent_value` | float | `0.0` | `0.0` | `0.0` |
| 右缩进单位下拉框 | `headings[n].right_indent_unit` | string | `"字符"` | `"字符"` | `"字符"` |
| 首行缩进输入框 | `headings[n].first_line_indent_chars` | float | `0.0` | `0.0` | `0.0` |
| **对齐区** | | | | | |
| 对齐方式单选组 | `headings[n].align` | string | `"LEFT"` | `"LEFT"` | `"LEFT"` |
| **空格区** | | | | | |
| 中英文间加空格复选框 | `headings[n].add_space` | bool | `false` | `false` | `false` |
| 空格数量微调框 | `headings[n].space_count` | int | `1` | `1` | `1` |

> 标题格式设置与正文格式设置的字段结构 **完全相同**，区别仅在默认值（标题默认粗体+左对齐+无首行缩进，正文默认两端对齐+首行缩进2字符）。

## 六、目录 (toc)

目录界面分为两个功能区：**目录标题设置**（顶部区域）和 **目录层级样式**（下方Tab页/列表）。

### 6.1 目录标题设置（TOC Title Settings）

> 对应 JSON 路径：`toc.*` 顶层字段

| 前端控件 | JSON字段 | 类型 | 示例值 | 说明 |
|---------|----------|------|--------|------|
| 启用目录格式化复选框 | `toc.enable` | bool | `true` | 是否启用目录格式化 |
| 目录标题文字文本框 | `toc.title_text` | string | `"目录"` | 目录标题文字 |

```

│  PySide6 UI 界面控件                  │
│  (QSpinBox / QComboBox / QCheckBox   │
│   QLineEdit / QRadioButton ...)       │

           ▼
```

## 十三、下拉框选项来源汇总

> 说明：软件界面中所有下拉框（QComboBox）/单选组（QRadioButton）/复选框（QCheckBox）的选项列表来源，主要分三类：**软件内置硬编码**、**运行时从系统获取**、**用户手动输入**。

### 13.1 字体类下拉框

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 中文字体下拉框 | `*.cn_font` | "宋体""黑体""仿宋""楷体"等 | **运行时从Windows系统获取**（通过Qt QFontDatabase查询已安装字体） |
| 英文字体下拉框 | `*.en_font` | "Times New Roman""Arial""Calibri"等 | **运行时从Windows系统获取**（同上） |
| 字号下拉框 | `*.size_cn` | "初号""小初""一号""小一""二号""小二""三号""小三""四号""小四""五号""小五"等 | **软件内置硬编码**（中文名→pt值对照表，符合国家标准字号规范） |

### 13.2 格式选项下拉框

| 控件 | JSON字段 | 选项列表 | 来源 |

| 左/右缩进单位下拉框 | `*.left_indent_unit` / `*.right_indent_unit` | "字符" / "厘米" / "磅" | **软件内置硬编码** |
| 对齐方式单选组/下拉框 | `*.align` | LEFT / CENTER / RIGHT / JUSTIFY | **软件内置硬编码**（对应Word对齐方式） |
| 中英文间加空格复选框 | `*.add_space` | true / false | **复选框（用户勾选）** |

### 13.10 来源分类统计

| 来源类型 | 覆盖字段数 | 典型字段 |
|---------|-----------|---------|
| **运行时从系统获取** | ~2类 | `cn_font`、`en_font`（所有模块的字体下拉框） |
| **软件内置硬编码** | ~12类 | 字号、行距模式、单位、对齐方式、编号方案、包装器、纸张大小、前导符、页码格式、表格样式等 |
| **用户手动输入（数值）** | ~10类 | 行距值、段前/段后值、缩进值、边距值、栏数、间距值等 |
| **用户手动输入（文字）** | ~3类 | 页眉文字、目录标题、自定义示例 |
| **复选框（用户勾选）** | ~20+个 | 粗体/斜体/下划线、预处理开关、应用开关、启用开关等 |

> **结论：约80%的下拉框选项来自软件内置硬编码，约10%来自运行时系统字体枚举，约10%由用户手动输入。软件不依赖外部配置文件提供下拉选项。**
