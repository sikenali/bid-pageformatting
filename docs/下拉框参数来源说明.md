# 下拉框参数来源说明

> 说明：本文档梳理软件界面中各下拉框（QComboBox）/单选组（QRadioButton）/复选框（QCheckBox）的选项列表来源。

---

## 一、来源分类

| 来源类型 | 说明 |
|---------|------|
| **软件内置硬编码** | 选项列表直接写在go 源码中编译到 .exe，不依赖外部文件 |
| **运行时从系统获取** | 软件启动时通过 Windows API / Qt API 查询当前系统环境 |
| **用户手动输入** | 由用户在输入框/微调框中填写 |

---

## 二、各模块下拉框来源明细

### 2.1 字体类

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 中文字体下拉框 | `*.cn_font` | "宋体""黑体""仿宋""楷体"等 | **运行时从Windows系统获取**（通过 Qt QFontDatabase 查询已安装字体） |
| 英文字体下拉框 | `*.en_font` | "Times New Roman""Arial""Calibri"等 | **运行时从Windows系统获取**（同上） |
| 字号下拉框 | `*.size_cn` | "初号""小初""一号""小一"…"八号" | **软件内置硬编码**（中文名→pt值对照表） |

### 2.2 格式选项

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 行距模式下拉框 | `*.line_spacing_mode` | SINGLE / 1.5倍 / DOUBLE / AT_LEAST / EXACT / MULTIPLE | **软件内置硬编码** |
| 段前/段后单位下拉框 | `*.space_*_unit` | "磅" / "行" | **软件内置硬编码** |
| 左/右缩进单位下拉框 | `*.left_indent_unit` / `*.right_indent_unit` | "字符" / "厘米" / "磅" | **软件内置硬编码** |
| 对齐方式 | `*.align` | LEFT / CENTER / RIGHT / JUSTIFY | **软件内置硬编码** |
| 中英文间加空格 | `*.add_space` | true / false | **复选框（用户勾选）** |
| 空格数量 | `*.space_count` | 1~9整数 | **微调框（用户手动输入）** |

### 2.3 标题编号规则

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 编号方案下拉框 | `rules[n].scheme` | ZH_NUM（中文数字）/ ARABIC（阿拉伯数字） | **软件内置硬编码** |
| 包装器下拉框 | `rules[n].wrapper` | DUNHAO（顿号）/ DOUBLE_PAREN（双括号）/ DOT（点）/ SINGLE_PAREN（单括号） | **软件内置硬编码** |
| 多层深度下拉框 | `rules[n].multi_depth` | "1.1" / "1.1.1" / "1.1.1.1" | **软件内置硬编码** |

### 2.4 表格

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 自动适应内容 | `table.autofit` | true / false | **复选框（用户勾选）** |
| 表格整体对齐 | `table.align` | LEFT / CENTER / RIGHT | **软件内置硬编码** |
| 单元格对齐 | `table.cell_align` | LEFT / CENTER / RIGHT | **软件内置硬编码** |
| 表格样式类型 | `table.style_type` | "默认样式"等 | **软件内置硬编码** |

### 2.5 目录

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 启用目录格式化 | `toc.enable` | true / false | **复选框（用户勾选）** |
| 目录标题文字 | `toc.title_text` | 任意文字 | **文本框（用户手动输入）** |
| 前导符样式 | `level_styles[n].tab_leader` | "点线（......）" / "虚线（----）" / "下划线（____）" / "无" | **软件内置硬编码** |
| 字体颜色 | `level_styles[n].color_rgb` | RGB颜色值 | **颜色选择器（用户选取）** |

### 2.6 页眉页脚

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 页眉文字 | `header_footer.header_text` | 任意文字 | **文本框（用户手动输入）** |
| 页眉/页脚对齐方式 | `*.header_align` / `*.footer_align` | "居中" / "左对齐" / "右对齐" | **软件内置硬编码** |
| 页眉下划线类型 | `header_footer.header_underline_type` | "不使用下划线" / "单下划线" / "双下划线"等 | **软件内置硬编码** |
| 页码格式类型 | `header_footer.footer_page_number_type` | "1, 2, 3, ..." / "i, ii, iii, ..." / "a, b, c, ..."等 | **软件内置硬编码** |
| 启用页眉/页脚 | `*.enable_header` / `*.enable_footer` | true / false | **复选框（用户勾选）** |

### 2.7 页面设置

| 控件 | JSON字段 | 选项列表 | 来源 |
|------|---------|---------|------|
| 纸张大小 | `margins.paper_size` | "A4" / "A3" / "B5" / "Letter"等 | **软件内置硬编码**（预设纸张规格→宽高对照表） |
| 上/下/左/右边距 | `margins.*_cm` | 任意数值 | **输入框（用户手动输入）** |
| 栏数 | `margins.columns` | 1~4整数 | **微调框（用户手动输入）** |
| 保持原纸张方向 | `margins.keep_original_orientation` | true / false | **复选框（用户勾选）** |

## 三、汇总统计

| 来源类型 | 覆盖范围 |
|---------|---------|
| **运行时从系统获取** | 中文字体列表、英文字体列表（2类字段，所有模块共享） |
| **软件内置硬编码** | 字号、行距模式、单位、对齐方式、编号方案、包装器、多层深度、纸张大小、前导符、页码格式、表格样式、制表符模式、下划线类型（~13类） |
| **用户手动输入（数值）** | 行距值、段前/段后值、缩进值、边距值、栏数、间距值、空格数量、检测间距（~8类） |
| **用户手动输入（文字）** | 页眉文字、目录标题、自定义示例（~3类） |
| **复选框（用户勾选）** | 粗体/斜体/下划线、预处理开关、应用开关、启用开关等（~20+个） |
